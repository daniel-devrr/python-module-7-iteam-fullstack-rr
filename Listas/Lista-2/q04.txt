ENUNCIADO:
Sem executar, determine o que cada print() imprime.
Explique a regra de escopo (LEGB) que determina cada resultado.

  x = 'global'

  def externa():
      x = 'enclosing'
      def interna():
          print(x)   # (a) imprime: ?
      interna()
      print(x)       # (b) imprime: ?

  externa()
  print(x)           # (c) imprime: ?

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
SUA RESPOSTA:

(a) imprime: enclosing

A função interna() procura a variável x seguindo a regra LEGB:

Local
Enclosing
Global
Built-in

Como não existe x dentro de interna(), o Python procura no escopo externo
(enclosing), que é a função externa(). Lá, x = 'enclosing'.

(b) imprime: enclosing

O print está dentro da função externa(), então ele utiliza a variável x
definida nesse próprio escopo:
x = 'enclosing'.

(c) imprime: global

Esse print está fora das funções, no escopo global do programa.
Assim, ele utiliza a variável:
x = 'global'.
